Member Enumeration Documentation

◆ Parse_Mode

enum colvarparse::Parse_Mode

How a keyword is parsed in a string.

Enumerator
parse_normal 

(default) Read the first instance of a keyword (if any), report its value, and print a warning when there is more than one

parse_silent 

Like parse_normal, but don't send any message to the log (useful e.g. in restart files when such messages are very numerous and redundant)
