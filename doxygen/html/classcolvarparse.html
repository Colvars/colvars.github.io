Member Enumeration Documentation

◆ Parse_Mode

enum colvarparse::Parse_Mode

How a keyword is parsed in a string.

Enumerator
parse_null 

Zero for all flags.

parse_echo 

Print the value of a keyword if it is given.

parse_echo_default 

Print the default value of a keyword, if it is NOT given.

parse_silent 

Do not print the keyword.

parse_required 

Raise error if the keyword is not provided.

parse_override 

Successive calls to get_keyval() will override the previous values when the keyword is not given any more

parse_restart 

The call is being executed from a read_restart() function.

parse_normal 

Alias for old default behavior (should be phased out)
